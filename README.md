# To-Do-App
A modern, responsive, and beginner-friendly to-do list built with pure HTML, CSS, and JavaScript. Designed for productivity and simplicity — with features like due dates, filters, and drag-and-drop reordering.

✨ Features
✅ Add, edit, and delete tasks
🌙 Toggleable dark mode
🕒 Set due dates for tasks
✅ Mark tasks as complete/incomplete
🔍 Filter by All / Active / Completed
🔃 Drag-and-drop task reordering
💾 Tasks saved in browser (localStorage)
📱 Fully responsive design (mobile-friendly)

🛠️ Tech Stack
  HTML5
  CSS3
  Vanilla JavaScript (no frameworks)\

🔗 [View the app here](https://ak5869.github.io/To-Do-App/)

📦 How to Use
Clone the repo or download the ZIP
Open index.html in your browser
Or deploy it via GitHub Pages
